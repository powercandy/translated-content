{{APIRef("DOM")}}

DOMString은 UTF-16 문자열입니다. JavaScript의 {{jsxref("String")}}도 UTF-16 문자열이기 때문에 DOMString은 String으로 연결됩니다.

DOMString을 받는 매개변수에 {{jsxref("null")}}을 전달하면, 보통 문자열로 변환해 "null"이 됩니다.
