{{APIRef("DOM")}}

USVString 对应 unicode 标量值的所有可能序列的集合。在 JavaScript 中返回时， USVString 映射到 {{JSxRef("String")}} 。它通常仅用于执行文本处理的 API，需要一串 unicode 标量值才能进行操作。除了不允许不成对的代理代码之外， USVString 等同于 {{DOMxRef("DOMString")}} 。 USVString 中存在的不成对的代理代码由浏览器转换为 Unicode '替换字符' U+FFFD, (�).

 

译注：

Unicode 标量值（ Unicode scalar values ）：字符的代号。

不成对的代理代码（ Unpaired surrogate codepoints ）：高代理代码后面没有低代理代码，或者低代理代码之前没有高代理代码。
