--- title: 伸缩性 slug: Glossary/Flex translation_of: Glossary/Flex ---

flex 是一个 CSS 的{{cssxref("display")}} 属性中新添加一个值。 随着inline-flex的使用，它将使它适用的元素成为一个{{glossary("flex container")}}（伸缩容器），而这个元素的每个子元素将成为 {{glossary("flex item")}}（伸缩项目）。伸缩项目将参与到 flex 布局中，所有由 CSS Flexible Box Layout Module（CSS 伸缩盒布局模型）定义的属性都能被它们使用。

flex 属性是flex-grow, flex-shrink 和 flex-basis 属性的简写。

此外， <flex> 可以作为弹性长度被引用在 CSS Grid（栅格）布局中。
