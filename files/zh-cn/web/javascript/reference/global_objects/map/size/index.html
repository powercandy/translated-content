size 是可访问属性，用于返回 一个{{jsxref("Map")}} 对象的成员数量。

{{EmbedInteractiveExample("pages/js/map-prototype-size.html")}}

描述

size 属性的值是一个整数，表示 Map 对象有多少个键值对。size 是只读属性，用 set 方法修改 size 返回 undefined，即不能改变它的值。

示例

var myMap = new Map();
myMap.set("a", "alpha");
myMap.set("b", "beta");
myMap.set("g", "gamma");

myMap.size // 3
