HTTP 协议的 508 Loop Detected 状态码可以在 WebDAV 协议（基于 Web 的分布式创作和版本控制）中给出。

508 码表示服务器中断一个操作，因为它在处理具有“Depth: infinity”的请求时遇到了一个无限循环。508 码表示整个操作失败。
