{{CSSRef}}

<dimension> CSS 数据类型是一个带有单位的{{CSSxRef("<number>")}}，例如：10px

CSS 使用度量值来描述距离 ({{CSSxRef("<length>")}})，时间 ({{CSSxRef("<time>")}})，频率 ({{CSSxRef("<frequency>")}})，分辨率 ({{CSSxRef("<resolution>")}}) 以及其他量。

格式

 <dimension> 的格式是一个{{CSSxRef("<number>")}}后紧随一个是标记符的单位，单位大小写不敏感。

Examples

合法的度量值

12px      12 像素
1rem      1 rem
1.2pt     1.2 点
2200ms    2200 毫秒
200hz     200 赫兹
200Hz     200 赫兹 (单位大小写不敏感)

非法的度量值

12 px       单位必须紧随数字，不能有空格
12"px"      单位是标识符，不能加引号
