{{CSSRef}}

CSS 数据类型 <angle-percentage> 代表了一种既可以是 {{Cssxref("angle")}} ，也可以是{{Cssxref("percentage")}}的数据类型

 

语法

有关此类型的详细文档，请参阅 {{Cssxref("angle")}} 和 {{Cssxref("percentage")}} 

在 calc() 中使用这个类型

当 <angle-percentage> 被正确定义后，它就代表了一个可以被 {{Cssxref("calc()")}} 表达式解析的角度
