小结

elements 返回一个 {{domxref("HTMLFormControlsCollection")}} ({{ HTMLVersionInline(4) }} HTMLCollection) 其中包含 FORM 的所有控件。需要注意的是，其中不包括 type 等于 image 的 input 元素。

你可以通过 name 或 id来访问对应的控件。

语法

nodeList = HTMLFormElement.elements

实例

var inputs = document.getElementById("form1").elements;
var inputByIndex = inputs[2];
var inputByName = inputs["login"];
