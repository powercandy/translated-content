audioBitsPerSecond  {{domxref("MediaRecorder")}} 接口的只读属性。它返回录制器所使用的音频编码码率. 或许与构造函数中指定的比特率有些不同（如果调用构造函数的时候有指定）。

语法

var audioBitsPerSecond = MediaRecorder.audioBitsPerSecond

值

 {{jsxref("Number")}} (无符号长整型).
