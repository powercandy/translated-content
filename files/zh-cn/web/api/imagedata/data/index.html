{{APIRef("Canvas API")}}

只读的 ImageData.data 属性，返回 {{jsxref("Uint8ClampedArray")}} ，描述一个一维数组，包含以 RGBA 顺序的数据，数据使用  0 至 255（包含）的整数表示。 

语法

imagedata.data

示例

var imagedata = new ImageData(100, 100);
imagedata.data; // Uint8ClampedArray[40000]
