{{ APIRef() }}

概要

返回一个可以检测 visibility 属性的 menubar 对象。

语法

objRef = window.menubar

示例

以下完整的 HTML 示例显示了使用 locationbar 对象的 visible 属性的方式。

<html>
<head>
  <title>Various DOM Tests</title>
  <script>
    var visible = window.menubar.visible;
  </script>
</head>
<body>
  <p>Various DOM Tests</p>
</body>
</html>
