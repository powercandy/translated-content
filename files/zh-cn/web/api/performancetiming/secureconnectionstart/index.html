概要

PerformanceTiming.secureConnectionStart 是一个返回代表一个时刻的 unsigned long long 型只读属性，为安全连接握手开始的时刻的 Unix 毫秒时间戳。如果只要你过的连接没有被请求，则它返回 0。

语法

time = performanceTiming.secureConnectionStart;
