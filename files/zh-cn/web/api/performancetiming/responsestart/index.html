概要

PerformanceTiming.responseStart 是一个返回代表一个时刻的 unsigned long long 型只读属性，为浏览器从服务器、缓存或者本地资源接收到响应的第一个字节之时的 Unix 毫秒时间戳。

语法

time = performanceTiming.responseStart;
