概要

PerformanceTiming.fetchStart 是一个返回代表一个时刻的 unsigned long long 型只读属性，为浏览器已经准备好去使用 HTTP 请求抓取文档之时的 Unix 毫秒时间戳。这一时刻在检查应用的缓存之前。

语法

time = performanceTiming.fetchStart;
