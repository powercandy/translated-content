{{AddonSidebar}}

在未来一个特定的时间运行的计划任务代码。这很像setTimeout()和setInterval()，不过这些函数仅可以按需使用而不能在后台页面工作。

想要使用这个 API，您需要获取"alarms"的permission。

类型

{{WebExtAPIRef("alarms.Alarm")}}
Information about a particular alarm.

方法

{{WebExtAPIRef("alarms.create()")}}
创建新的 alarm.
{{WebExtAPIRef("alarms.get()")}}
通过名称获取指定的 alarm.
{{WebExtAPIRef("alarms.getAll()")}}
获取所有的 alarm.
{{WebExtAPIRef("alarms.clear()")}}
清除指定名称的 alarm.
{{WebExtAPIRef("alarms.clearAll()")}}
清除所有的 alarm.

事件

{{WebExtAPIRef("alarms.onAlarm")}}
当 alarm 发生的时候触发。
