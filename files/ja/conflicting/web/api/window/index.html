この window で現在の document が画面に描画された回数を返します。

 構文

var paintCount = window.mozPaintCount;
  • paintCount は window.mozPaintCount プロパティの値を保存します。
  • window.mozPaintCount の値は long long です、そして最初に document が生成された時にはゼロから開始し、 document が描画される度に1増加します。

仕様

W3C の技術仕様、または勧告の一部ではありません。
