_`lang`_ メンバーは単一の[言語タグ](../HTML/Global_attributes/lang)を持つ文字列です。これは、マニフェストの書字方向に対応しているメンバーの主言語を指定し、 [`dir`](/ja/docs/Web/Manifest/dir) と共に書字方向を特定します。

## 例

```json
"lang": "en-US"
```
