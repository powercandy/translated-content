概要

dialogArguments プロパティは、 {{domxref("window.showModalDialog()")}} メソッドに渡されたパラメータを返します。モーダルダイアログボックスの作成時に、引数にどのようなパラメータが指定されたを判断することができます。

構文

value = window.dialogArguments;
