{{APIRef("Encoding API")}}

**`TextEncoder.encoding`** は読み取り専用プロパティで、特定のエンコーダーで使用されるエンコーディングアルゴリズムの名前が入った文字列を返します。

持つことができる値は `utf-8` のみです。

## 値

文字列です。
