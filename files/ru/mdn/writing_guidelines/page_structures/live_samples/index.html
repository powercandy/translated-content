MDN имеет систему «живых примеров», где пример кода, показанный на странице, служит для отображения вывода этого же примера. Однако во многих существующих статьях есть примеры кода, которые ещё не используют эту систему, и их необходимо преобразовать.

Живые примеры, позволяющие продемонстрировать работу кода, делают документацию более динамичной и наглядной. В этом руководстве описывается, как "оживить" существующие примеры.

Где это необходимо сделать?
В статьях с тегом NeedsLiveSample
Что вам нужно знать, чтобы начать?
  • Понимание HTML, CSS и/или JavaScript, в зависимости от примера кода.
  • Способность использовать макросы KumaScript в статьях MDN.
Что нужно для этого сделать?
  1. Выберите статью из списка статей с тегами NeedsLiveSample, где есть пример кода, с понятной, для вас функцией.
  2. Преобразуйте пример кода, чтоб он стал "живым".
  3. Удалите любой код или изображение, которые использовались для наглядного примера вывода результата работы кода.

Подробнее об создании и редактировании живых примеров, см. Использование системы живых примеров
