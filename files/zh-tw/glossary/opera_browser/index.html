Opera 是最常使用的 {{glossary("browser")}} 第五名，1996 年正式推出，一開始僅能在 Windows 使用。Opera 自 2013 年起使用 {{glossary("Blink")}} 排版引擎，在那之前使用的是 {{glossary("Presto")}}。Opera 也有提供行動與平板電腦版本。
