{{APIRef("DOM")}}

DOMString 是編碼為 UTF-16 的字串。當 JavaScript 使用這類字串時， DOMString 會映射成一個 {{jsxref("String")}}.

傳送 null 給接受 DOMString 的 method 或參數時，會將其轉換成 "null"。
